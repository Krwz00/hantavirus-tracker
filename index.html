Méthodologie
Le bandeau de cas (carte) recense uniquement les cas confirmés en laboratoire et notifiés par les autorités sanitaires nationales ou internationales (OMS, ECDC, ministères de la Santé). Chaque cas est rattaché à une source publique vérifiable. Les actualités sont agrégées toutes les heures depuis les flux RSS officiels (OMS Disease Outbreak News, Santé publique France, ECDC, info.gouv.fr, ANRS MIE). Les factchecks sont générés automatiquement à partir de contenus repérés sur des sources connues pour relayer la désinfox, puis vérifiés contre la base de référence OMS / Institut Pasteur via un modèle de langage. Toute erreur peut être signalée.
